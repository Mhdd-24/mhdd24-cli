CLI commands & examples

⌘

Global npm package mhdd24-cli. Run these from your project root (where package.json or git lives). Command name: mhdd24.

Install
npm install -g mhdd24-cli

Core

push

Git add · commit · push current branch. Runs optional Vercel prod deploy if .mhdd24rc sets vercel.cliDeployAfterPush.

mhdd24 push "fix: handle edge case"
deploy

Same as push, then runs build and deploy scripts from .mhdd24rc (defaults npm run build / npm run deploy).

mhdd24 deploy "chore: release candidate"
release

Requires clean git tree. Runs standard-version, pushes branch and tags, then npm publish (asks for OTP).

mhdd24 release
version

Bumps package.json via npm version. Use auto to infer patch/minor/major from conventional commits since last tag. Add --no-git-tag-version to skip git tag.

mhdd24 version patch
mhdd24 version auto
mhdd24 version minor --no-git-tag-version

Vercel

vercel-setup

Pick GitHub repo (gh or manual owner/repo), optional origin, vercel link, vercel git connect, updates .mhdd24rc.

mhdd24 vercel-setup
vercel

Production deploy now (vercel deploy --prod --yes). Needs Vercel CLI logged in.

mhdd24 vercel
domain-add

Adds a domain to the linked Vercel project; appends to vercel.domains in .mhdd24rc.

mhdd24 domain-add app.example.com

Project insight

detect

Detects framework / package manager, writes framework block to .mhdd24rc; optional prompt to sync build.

mhdd24 detect
dashboard

Local web UI on 127.0.0.1 (default port 3847). Env MHDD_DASH_PORT or pass port as argument.

mhdd24 dashboard
mhdd24 dashboard 9000

Dev

doctor

Node, npm, git versions and binary paths; MONGO_URI; detected framework when package.json exists.

mhdd24 doctor
logs

Tails logs/app.log (Ctrl+C to exit).

mhdd24 logs
analyze

Bundle size via npx source-map-explorer build/static/js/*.js (CRA-style paths).

mhdd24 analyze
rollback

Soft reset last commit (git reset --soft HEAD~1).

mhdd24 rollback
smart-commit

Infers message from staged paths, commit, push; optional Vercel like push.

mhdd24 smart-commit

Environment

env

Appends KEY=value to .env in the current directory.

mhdd24 env API_URL https://api.example.com

Git

branch · switch · pull · status · clean-git

Create branch, checkout, pull current branch, short status, hard reset + clean untracked.

mhdd24 branch feature/oauth
mhdd24 switch main
mhdd24 pull
mhdd24 status
mhdd24 clean-git

Project

init

Writes .mhdd24rc (with framework detection if package.json exists) and creates .env if missing.

mhdd24 init
open

Opens current folder in VS Code (code .).

mhdd24 open
open-url

Opens a URL with the system default handler (macOS open).

mhdd24 open-url https://github.com/you/repo

Build & deploy scripts

build · deploy-only

Runs build / deploy from .mhdd24rc or falls back to npm run build / npm run deploy.

mhdd24 build
mhdd24 deploy-only
clean

Removes node_modules and lockfile, then npm install.

mhdd24 clean

Other

seed

MongoDB seed helper for a specific api/…/SeedData layout; requires MONGO_URI. Locales optional, default en,de,ta,ml.

mhdd24 seed
mhdd24 seed en,de
Help

Prints all commands when the first argument is missing or unknown.

mhdd24
mhdd24 help
